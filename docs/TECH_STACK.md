# Tech Stack

- **Next.js:** For building the React application with Server-Side Rendering (SSR) and optimized performance.
- **TypeScript:** For type safety and better developer experience.
- **Tailwind CSS:** For rapid, utility-first styling.
- **Framer Motion:** For smooth, physics-based animations of SVG elements in the visualizer.
- **shadcn/ui:** For beautiful, accessible, and customizable UI components.
- **Lucide React:** For icons.
